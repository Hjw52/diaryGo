# diaryGo

![](https://s3.bmp.ovh/imgs/2022/04/16/160571800512b709.png)

一款基于electron+Vue3开发的桌面端便签应用，拟态UI风格。

## 应用功能

- [x] 新建与编辑待办事项
- [x] 恢复与删除已完成事项
- [x] 查看历史完结事项
- [x] 屏幕锁定
- [x] 一键换肤

## 技术栈

- electron
- Vue3
- element-plus
- lowdb
